La estrategia de producto

De todo el análisis salen estas conclusiones claras:

  • Producto fusionado — en vez de PULSO e IMPULSO como 2 informes largos, una experiencia única adaptada al perfil del inversor
  • Freemium — precio de entrada bajísimo (29€) para que entre mucha gente. La gente ve el potencial gratis y paga para desbloquear los datos concretos
  • MTVV como diferenciador — nadie más tiene datos de campo de inversores formados. Cada MTVV que entra mejora todo el sistema
  • Portal web moderno — adiós Notion. Link compartible, PDF exportable, diseño tech que impacta
  • Captación MTVV — sistema de incentivos: aportas un MTVV → recibes un informe gratis. Dos tipos de proveedor: el que hace operaciones puntuales y el profesional que hace MTVVs como servicio
  • Rama de oportunidades — lo que ya hacemos con JT pero escalado: validar deals con sello IM·PULSO antes de presentarlos a inversores

Dónde estoy yo con todo esto
